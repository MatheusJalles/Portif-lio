# Portfólio - Matheus Jalles

Projeto estático simples com HTML e CSS que representa o portfólio do Matheus Jalles.

## Estrutura
- `opção 2/` — versão de trabalho atual do site (conteúdo principal)
- `docs/` — (criado automaticamente) contém a versão pronta para GitHub Pages

## Como testar localmente
1. Abra `docs/index.html` (ou `opção 2/index.html`) diretamente no navegador.
2. Ou use uma extensão `Live Server` no VS Code para visualizar com reload ao salvar.

## Publicação com GitHub Pages
1. Vá em Settings → Pages no GitHub.
2. Selecione a branch `main` e a pasta `/docs` como fonte (ou deixe `/` se mover o site para a raiz).
3. Salve: o site ficará disponível em `https://<seu-usuario>.github.io/<repo>/`.

## Observações
- Atualize os links de contato no HTML (`wa.me`, `mailto`) para seus dados reais.
- Se preferir, posso adicionar um workflow GitHub Action para deploy automático.
